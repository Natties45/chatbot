# OLS Chatbot Deploy — FFX Squad Style

> แผนสั้น — แต่ละ Phase = Mission, แต่ละ Skill = สมาชิกในทีม

---

## 🎯 Phase 0: Prepare (สำรวจ + ตระเตรียม)

**บท:** ก่อนออกเดินทาง — ดูแผนที่, เช็คของในกระเป๋า, ถามลูกค้าว่าต้องการอะไร

**Action:**
1. เช็ค Skill ทีมี → map กับ phase ที่ต้องใช้
2. เช็ค MCP tools ทีมี → map กับ phase
3. **ห้าม load skill จนกว่าจะถึง phase นั้น**
4. หา Requirements จากคุณ natti ก่อนเริ่ม

**Requirements ที่ต้องขอคุณ:**

| # | Requirement | ใครตอบ | ได้/ยัง? |
|---|-------------|--------|---------|
| 1 | Server Provider + IP + SSH key | คุณ natti | ❌ |
| 2 | Ollama Cloud Pro API key | คุณ natti | ❌ |
| 3 | LINE Notify token (หรือใช้ email) | คุณ natti | ❌ |
| 4 | Domain name? หรือ IP-only? | คุณ natti | ❌ |
| 5 | ต้อง backup ไว้ที่ remote หรือ local พอ? | คุณ natti | ❌ |

**Squad สมาชิกที่ใช้ใน Phase นี้:**
- ❌ `plan` — กำลังใช้ เขียนแผน
- ❌ tools — ยังไม่เรียก ดูอย่างเดียว

**บท:** Tidus + Yuna — เปิดประตูสู่ Spira (server)

**Mission:** เตรียม server + ขึ้น Ollama + Dify

**Squad:**
| สมาชิก | Skill / Tool | บทบาท |
|--------|-------------|-------|
| Tidus | `terminal` | **ตัวถล่มหลัก** — SSH, Docker, script |
| Yuna | `browser` | **เรียก Aeon** — verify Dify UI จริง |
| Lulu | `systematic-debugging` | **เวทมนตร์รับมือ** — เผื่อ Docker fail |

**User Requirements ที่ใช้:**
- Server IP + SSH key ✓
- `.env` secrets (DIFY_SECRET_KEY, DB passwords)

**Model: 🔹 deepseek-v4-flash**
